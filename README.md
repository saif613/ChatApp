# ChatApp Backend API

A real-time chat application backend built with ASP.NET Core using Clean Architecture, CQRS, MediatR, SignalR, Entity Framework Core, and JWT Authentication.

## Features

* JWT Authentication & Authorization
* Real-time messaging using SignalR
* Private conversations between users
* Message editing and deleting
* Seen messages system
* Typing indicator
* User online/offline presence system
* Last seen tracking
* Conversation management
* Search messages
* Search users
* CQRS with MediatR
* FluentValidation
* Global Exception Middleware
* Request Logging Middleware
* Clean Architecture
* Repository-less Service Pattern
* SQL Server & Entity Framework Core

## Technologies

* ASP.NET Core Web API
* SignalR
* Entity Framework Core
* SQL Server
* MediatR
* FluentValidation
* AutoMapper
* JWT Authentication

## Architecture

The project follows Clean Architecture principles:

* API Layer
* Application Layer
* Domain Layer
* Infrastructure Layer
